# Netflix SQL Data Analysis

This project analyzes the Netflix dataset using SQL to explore content distribution across countries, directors, and release years.

## Dataset
Netflix titles dataset from Kaggle.

## SQL Analysis Performed

1. Count Movies vs TV Shows
2. Top countries producing Netflix content
3. Directors who created both Movies and TV Shows
4. Countries producing more Movies than TV Shows
5. Content distribution by release year

## SQL Concepts Used

- GROUP BY
- COUNT
- COUNT DISTINCT
- HAVING
- CASE WHEN
- ORDER BY

## Tools Used

- SQL Server
- GitHub
